# 付録A：四位一体コミットメント契約書（テンプレート）

---

## 使い方

この契約書テンプレートは、PM育成プログラムを開始する際に、教育部門・経営層・現場・採用部門の4者が署名するために使用します。

### 記入方法

1. 【　】内に、あなたの会社の情報を記入してください
2. 数値目標は、あなたの会社の状況に合わせて調整してください
3. 4者が署名し、原本を保管してください
4. コピーを、社内に掲示してください

レベル3：3年後、レベル3認定者が0名
→ PM育成プログラムの抜本的見直し、または撤退

経営層は、以下の条件で投資を継続する。

1. 1年目に成果が出なくても、2年目の投資を継続する
2. 2年目に成果が出なくても、3年目の投資を継続する
3. 3年目の最終成果で、継続可否を判断する

経営層は、PM育成プログラムにおける失敗を許容する。

許容する失敗：
・PM候補者の案件赤字（上限【500万円】/件）
・顧客クレーム（重大事故を除く）
・納期遅延（【1ヶ月】以内）

失敗リカバリ予算：年間【300万円】を確保

失敗による評価への影響：
・PM候補者の人事評価に反映しない
・現場部門の評価に反映しない

第7条：現場協力への指示

経営層は、現場部門に対し、以下を指示する。

1. 各部門は、年間【2名】のPM候補者を受け入れる
2. PM候補者の稼働率を【80%】に制限する
3. 現場メンターによる月【2回】のメンタリングを義務化する
4. 育成協力を、マネージャー評価の【20%】に組み込む

現場部門が協力を拒否した場合、経営層が直接介入する。

第8条：撤退基準

経営層は、以下の場合、PM育成プログラムを撤退する。

1. 離脱率が【50%】を超えた場合（3年間累計）
2. 教育部門が数値責任を放棄した場合
3. 現場が育成協力を拒否し、改善されない場合
4. 3年後の最終評価で、レベル3認定者が【0名】の場合

撤退判断は、経営層が行う。

第9条：PM候補者の受入義務

現場部門は、PM育成プログラムにおいて、以下を約束する。

PM候補者の受入人数：年間【2名】（3年間で計【6名】）
受入期間：1名あたり【1〜2年間】
受入拒否の禁止：「忙しい」を理由とした受入拒否は認めない

第10条：稼働率制限の義務

現場部門は、PM候補者の稼働率を以下に制限する。

PM候補者の稼働率：【80%】以下
残り【20%】の時間：育成活動に充てる
　・研修受講
　・自己学習
　・メンタリング
　・育成案件の振り返り

稼働率超過の禁止：
PM候補者を【120%】稼働させることは禁止する。
違反した場合、ペナルティを科す。

第11条：メンタリングの義務

現場部門は、PM候補者に対し、以下のメンタリングを実施する。

メンタリング頻度：月【2回】、1回【30分】
メンタリング内容：
　・案件の進捗確認
　・課題の相談
　・成功・失敗の振り返り
　・次のステップの設定

メンタリング実施率：【100%】を目標とする
3ヶ月連続で未実施の場合、ペナルティを科す。

第12条：育成案件の提供義務

現場部門は、PM候補者に対し、以下の育成案件を提供する。

育成案件の数：年間【2〜3件】
育成案件の難易度：PM候補者のレベル+1
　・レベル1候補者 → レベル2相当の案件
　・レベル2候補者 → レベル3相当の案件

育成案件の特徴：
　・失敗しても、顧客との関係が壊れない案件
　・PM候補者が主担当として経験できる案件
　・適度なチャレンジがある案件

第13条：失敗時のフォロー義務

PM候補者が案件で失敗した場合、現場部門は以下を行う。

リカバリ支援：
　・現場マネージャーが案件に介入し、リカバリを支援
　・顧客への説明・謝罪を、マネージャーが行う
　・PM候補者を責めず、再チャレンジの機会を与える

失敗の振り返り：
　・失敗原因の分析
　・再発防止策の立案
　・次の案件での改善

評価への非反映：
PM候補者の失敗を、人事評価に反映しない。

第14条：育成協力の評価

現場部門の育成協力は、以下の基準で評価される。

評価項目：
1. PM候補者受入数（5%）
2. メンタリング実施率（5%）
3. PM候補者のレベルアップ（5%）
4. 育成案件の提供（5%）

評価結果の反映：
・賞与
・昇進
・表彰
・チーム報奨

第15条：育成拒否のペナルティ

以下の行為は、育成拒否と見なし、ペナルティを科す。

育成拒否の行為：
・PM候補者の受入拒否
・メンタリングの未実施（3ヶ月連続）
・育成案件の未提供
・PM候補者への過剰負荷（稼働率120%）
・失敗時の責任追及

ペナルティ：
・レベル1：警告（1回目）
・レベル2：経営層からの直接指導（2回目）
・レベル3：人事評価への反映（3回目）
・レベル4：マネージャー職の解任（4回目）
